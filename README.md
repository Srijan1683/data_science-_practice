# Data Science Practice

This repository is a collection of my data science practice work. It includes dataset-based analysis, exploratory work in Jupyter notebooks, basic data cleaning, summary statistics, and visualizations.

The repo is designed to grow over time. More CSV files, notebooks, analyses, and visual reports will be added as I continue practicing with different datasets and questions.

## Current Contents

### Student Performance Analysis

Path: `Data Analysis/StudentPerformance/`

This section currently includes:

- `StudentPerformanceFactors.csv` for the dataset
- `analysis.ipynb` for exploratory analysis and visualization

The notebook currently covers:

- inspecting the dataset structure and shape
- using descriptive statistics for numeric and categorical columns
- adjusting record indexing for easier reading
- calculating average study hours
- analyzing parental involvement patterns
- comparing parental involvement with exam score categories
- connecting the dataset to MySQL for SQL-based exploration
- creating visualizations with libraries such as Matplotlib and Seaborn
